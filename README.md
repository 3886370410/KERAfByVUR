# 项目简介

## 核心功能
- 基于 YOLOv8 的水面垃圾检测系统，已训练好模型
- 支持 5 类垃圾检测：['can', 'foam', 'plastic', 'plastic bottle', 'unknow']
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供完整标注数据集（3600 张图片）
- 包含训练好的模型及训练结果
- 提供项目运行教程和完整程序文件
- 支持自定义置信度和 IOU 阈值
- 支持选择显示标签和原图
- 支持批量检测并查看所有结果
- 支持检测结果保存
- 提供图形用户界面（UI）

## 适用场景
- 水域环境监测与保护
- 水面垃圾自动识别与统计
- 环保监测与研究
- 水质管理相关应用
- 计算机视觉学习与实验

## 技术特点
- 基于 Python 3.9 开发
- 使用 OpenCV、PyQt5、Ultralytics、Torch 等技术栈
- 采用 YOLOv8 作为检测模型
- 提供完整的开发环境和运行环境
- 包含 3600 张标注数据集
